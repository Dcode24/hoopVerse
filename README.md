
# Basketball Blog App

hoopVerse is a full-stack basketball blog app. Users can create an account and log in to their account where they can view other user's blogs and also create and edit their own blogs.


## Frontend

- HTML
- CSS
- JavaScript
- React

## Backend

- Nodejs
- Express
- MongoDB
